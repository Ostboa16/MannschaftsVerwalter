In diesem Programm können sie Ihre eigene Fußballmannschaft erstellen.
Sie können Spieler hinzufügen, entfernen und Rückennummern ändern.
Die Spieler werden immer nach der Rückennummer sortiert.
Spieler können entfernt werden indem Sie die Rückennummer vom Spieler den Sie entfernen wollen
eingeben.
Außerdem können sie ihre Mannschaft in einer Datenbank und in einer .bin Datei speichern.
